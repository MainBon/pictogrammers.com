### Get Admin Package Fonts

```
GET api/admin/package/:package_id/font
```

<Tabs>
  <Tab title='Description'>
    Retrieve an array of fonts for a given package.
  </Tab>
  <Tab title='Response'>
    ```yaml-to-json
    type: array
    items:
      $ref: '#/contribute/api/models/font'
    ```
  </Tab>
</Tabs>
