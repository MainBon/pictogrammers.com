### Get Icon by Id

```
GET /icon/:iconId
```

<Tabs>
  <Tab title='Description'>
    Retrieve a single icon and all public properties at once. Note looping this is very slow as it uses no cache layer to ensure the latest data.
  </Tab>
  <Tab title='Response'>
    ```yaml-to-json
    $ref: '#/contribute/api/models/icon'
    ```
  </Tab>
</Tabs>
