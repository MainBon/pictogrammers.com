### Post Admin Icon Name

```
POST /admin/icon/name
```

<Tabs>
  <Tab title='Description'>
    Update `icon.name` to rename an icon.
  </Tab>
  <Tab title='Request'>
    ```yaml-to-json
    type: object
    properties:
      icon:
        type: object
        properties:
          id:
            type: string
            format: uuid
          name:
            type: string
            example: new-name
    ```
  </Tab>
  <Tab title='Response'>
    ```yaml-to-json
    $ref: '#/contribute/api/models/icon'
    ```
  </Tab>
</Tabs>
