### Post Admin User

```
POST /admin/user
```

<Tabs>
  <Tab title='Description'>
    Insert a new `user` into the database. These users are readonly and cannot be logged into.

    <Note type='warning'>This API requires elevated `core` access.</Note>
  </Tab>
  <Tab title='Request'>
    ```yaml-to-json
    type: object
    properties:
      user:
        type: object
        properties:
          name:
            type: string
          github:
            type: string
          email:
            type: string
            format: email
          description:
            type: string
          links:
            type: json
          contributor:
            type: boolean
          core:
            type: boolean
    ```
  </Tab>
  <Tab title='Response'>
    ```yaml-to-json
    $ref: '#/contribute/api/models/user'
    ```
  </Tab>
</Tabs>
