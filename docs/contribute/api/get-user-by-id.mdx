### Get User by Id

```
GET api/user/:userId
GET api/package/:packageId/user/:userId
```

<Tabs>
  <Tab title='Description'>
    Retrieve a single user (aka contributor). Using the alternative url with `packageId` will filter the `iconCount` to a specific `packageId`.
  </Tab>
  <Tab title='Response'>
    ```yaml
    $ref: '#/api/model/user'
    ```
  </Tab>
</Tabs>
