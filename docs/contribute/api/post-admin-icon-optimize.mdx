### Post Admin Icon Optimize

```
POST api/admin/icon/optimize
```

<Tabs>
  <Tab title='Description'>
    Optimize `icon.data` with SVGO. This is used to preview the change and will not touch the database.
  </Tab>
  <Tab title='Request'>
    ```yaml
    type: object
    properties:
      icon:
        type: object
        properties:
          id:
            type: string
            format: uuid
          data:
            type: string
            example: M...Z
    ```
  </Tab>
  <Tab title='Response'>
    ```yaml
    $ref: '#/api/model/icon'
    ```
  </Tab>
</Tabs>
