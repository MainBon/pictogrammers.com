### Get List of Users by PackageId

```
GET /package/:packageId/user
```

<Tabs>
  <Tab title='Description'>
    List all `user` objects associated to a `packageId`.
  </Tab>
  <Tab title='Response'>
    ```yaml-to-json
    type: array
    items:
      $ref: '#/contribute/api/models/user'
    ```
  </Tab>
</Tabs>
