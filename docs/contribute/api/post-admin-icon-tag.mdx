### Post Admin Icon Tag

```
POST api/admin/icon/tag
```

<Tabs>
  <Tab title='Description'>
    Inserts `tag` into the database for an icon.
  </Tab>
  <Tab title='Request'>
    ```yaml
    type: object
    properties:
      icon:
        type: object
        properties:
          id:
            type: string
            format: uuid
      tag:
        type: object
        properties:
          name:
            type: string
            example: user
    ```
  </Tab>
  <Tab title='Response'>
    ```yaml
    $ref: '#/api/model/icon'
    ```
  </Tab>
</Tabs>
