The [Memory Icons](https://www.figma.com/community/plugin/1234987873589988746) Figma plugin allows you to easily search and drop-in any of our icons directly into your designs.

<Button href='https://www.figma.com/community/plugin/1234987873589988746' startIcon='si:figma' variant='contained'>Get the Memory Icons Figma Plugin Now</Button>

## How To Use

1. Simply "Run" the plugin and you will be presented with a library view for Memory Icons.
2. Use the Search field to narrow your view to only icons you are interested in.
3. Simply click an icon to add it directly into your Figma design!
